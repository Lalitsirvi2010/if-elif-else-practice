# If-Elif-Else Practice

Python programs to practice conditional statements (`if`, `elif`, `else`).

## Programs

| File | Description |
|------|-------------|
| `atm_checker.py` | Validates ATM withdrawals (balance, amount, multiples of 100) |
| `bill_calculator.py` | Applies discounts based on total bill amount |
| `bmi_calculator.py` | Classifies weight using BMI ranges |
| `calculator.py` | Basic calculator with divide-by-zero handling |
| `electricity_bill.py` | Calculates electricity bill using tiered slab rates |
| `equation.py` | Solves quadratic equations and handles non-real roots |
| `grade_calculator.py` | Calculates grades from subject marks |
| `mood_suggestion.py` | Suggests music or activities based on mood |
| `movie_ticket.py` | Calculates movie ticket price by age and popcorn |
| `number_guess.py` | Guessing game with higher/lower hints |
| `leap_year_checker.py` | Checks whether a year is a leap year using 4/100/400 rules |
| `rock_paper_scissors.py` | Rock Paper Scissors game against the computer |
| `temperature_advice.py` | Gives weather advice based on temperature |
| `traffic_fine.py` | Calculates traffic fines for overspeeding and license |
| `user_login.py` | Login system with separate username/password errors |
| `blood_donation_eligibility.py` | Blood donation eligibility based on age, weight, gender, and days since the last donation |

## How to run

Each program runs on its own. From this folder:

```bash
python atm_checker.py
python bill_calculator.py
# ... and so on for any file
```

## Requirements

- Python 3
- No extra libraries needed (`random` and `math` are built-in)
